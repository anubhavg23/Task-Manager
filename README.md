<h1>📋 SETMYTASK - PanScience</h1>

<p>
A modern and user-friendly <b>Task Management Web App</b>.<br>
Built to help <b>admins assign work</b> and <b>users complete & submit tasks with ease</b>.<br>
Stay organized, track progress, and improve team collaboration — all in one place.
</p>

<hr>

<h2>✨ Features</h2>

<h3>🔑 Authentication</h3>
<ul>
  <li>Secure <b>login system</b> for both <b>Admins</b> and <b>Users</b></li>
  <li>Separate dashboards based on roles</li>
</ul>

<h3>👨‍💼 Admin Dashboard</h3>
<ul>
  <li>➕ Create and assign new tasks</li>
  <li>🎯 Assign tasks to specific users</li>
  <li>✅ Track who has completed work</li>
  <li>📝 Provide feedback and monitor progress</li>
</ul>

<h3>👩‍💻 User Dashboard</h3>
<ul>
  <li>📂 View assigned tasks in real-time</li>
  <li>⬆️ Upload submissions (supports <b>PDFs</b>)</li>
  <li>📊 Get feedback and see task status updates</li>
</ul>

<h3>📊 Progress Tracking</h3>
<ul>
  <li>Clear & interactive charts for task updates</li>
  <li>Easy reporting for both Admins and Users</li>
</ul>

<h2>📌 How to Run Locally</h2>

<pre>
# Clone the repo
git clone https://github.com/anubhavg23/Task-Manager.git

# Go to Backend
cd Backend
npm install
npm start

# Open new terminal → Go to Frontend
cd Frontend
npm install
npm start
</pre>

<p>
App will run on:<br>
- Frontend → <code>http://localhost:3000</code><br>
- Backend → <code>http://localhost:5000</code>
</p>
